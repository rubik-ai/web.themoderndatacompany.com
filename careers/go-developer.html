We are looking for multiple Go Lang Developers with experience in building high-performing, scalable, enterprise-grade applications.

You will be part of a talented software team that works on mission-critical applications. Go developer roles and responsibilities include managing application development while providing expertise in the full software development lifecycle, from concept, design and testing.

Roles & Responsibilities

  • Writing scalable, robust, testable, efficient, and easily maintainable code.
  • Translating software requirements into stable, working, high-performance software.
  • Playing a key role in architectural and design decisions and towards building an efficient microservice distributed architecture.

Requirements

  • Degree in Computer Science.
  • Strong knowledge of Go programming language, paradigms, constructs, and idioms.
  • Knowledge of common Go routine and channel patterns.
  • Experience with the full site of Go frameworks and tools, including:
  • Dependency management tools such as Godep, Sltr, etc.
  • Go’s templating language
  • Go’s code generation tools, such as Stringer(X)
  • Popular Go web frameworks, such as Revel, GIN
  • Router packages, such as Gorilla Mux
  • Design Pattern.
  • Ability to write clean and effective Godoc comments.
  • Familiarity with code versioning tools (such as Git, SVN, and Mercurial).
  • Proficient with testing viz. Unit testing, Integration testing.
  • Experience in application design using design patterns.

Location

  • Indore, Madhya Pradesh, India
  • Hyderabad, Telangana, India
  • Open for Remote full-time role.
